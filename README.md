# Kana Dojo

A browser-based drill for learning Japanese hiragana and katakana, with shape-based mnemonics to help the characters stick.

## Features

- **Drill mode** — type the romaji for each character, get instant feedback
- **Study mode** — browse all kana with mnemonics, click any card to expand
- **Hint system** — press Space to reveal the romaji, a mnemonic, and hear the character spoken aloud (toggleable); audio uses the browser's built-in Web Speech API with a Japanese voice — no external service or files required
- **Cheatsheet** — side-by-side hiragana / katakana / romaji reference
- Toggles for Hiragana, Katakana, or Both
- Optional Dakuten / Handakuten and Yōon (combination) character sets
- Score and streak tracking

## Usage

Open `index.html` in any browser — no build step, no dependencies.

Or visit the live version: [indygwyn.github.io/kana-dojo](https://indygwyn.github.io/kana-dojo)
